# local-development
1) install node 8.9 (once)
2) install yarn (once)
3) yarn (after fetching new version)
4) yarn open:src (starts a development server)
     or
   yarn open:dist (compiles and opens a production build)


# building a dist
   1) yarn
   2) yarn build
